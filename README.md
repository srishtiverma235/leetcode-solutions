# LeetCode Solutions

This repository contains my LeetCode solutions in Java, organized by topic and problem number.

The goal of this repository is to:

- Strengthen Data Structures and Algorithms concepts
- Prepare for coding interviews and placements
- Practice problem-solving and pattern recognition
- Maintain a structured collection of solutions for future revision

## Language

- Java

## Topics Covered

- Arrays
- Strings
- Hashing
- Two Pointers
- Sliding Window
- Linked List
- Stack
- Queue
- Binary Search
- Trees
- Binary Search Trees
- Heaps
- Graphs
- Dynamic Programming
- Greedy Algorithms
- Backtracking
